checkLogin()

checkLogin(  $username,   $password) : boolean

Double checks that the Student's login information is correct

It is given a username and password variables as checkers. The passed variables are then used to grab a student_id (using a DB method), and if that fails, then there is no user with that username or password. If it finds an ID that is associated with the passed args, Then it trys to get a table row with all the Student information. It uses the information in that Database row to see if it matches the passed args. If it does, then that student is logged in. If it does not, then the information given is incorrect.

Parameters

$username
$password

Returns

boolean —

(true on success, false on fail)
