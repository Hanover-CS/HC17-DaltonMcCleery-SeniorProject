connect()

connect() : \the

This is the main method of the DB Class.

When this method is called (and it can be called statically without referencing an object), it checks an instance property, above, to see if there was a DB connection previously made. If there was, then there is no need to create another connection, just return the first (and only) connection. If there was not, then it (and only it) calls the construct method of this class and creates a DB connection.

Returns

\the —

mysqli connection object
